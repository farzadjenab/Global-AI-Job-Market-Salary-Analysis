# Global AI Job Market & Salary Analysis

This project provides an in-depth analysis of global AI job market trends, salaries, and key job attributes based on two large datasets containing detailed information about job listings in the AI industry. The analysis focuses on various aspects such as job titles, salaries, experience levels, education requirements, company locations, remote work ratios, and more.

## Project Overview

The datasets contain information on job roles in the AI sector across multiple countries. The analysis includes:

- **Salary Distribution**: Comparing salaries across job titles, locations, and experience levels.
- **Experience vs. Salary**: Investigating the relationship between years of experience and salary.
- **Job Locations**: Understanding how job location impacts salary.
- **Skills**: Analyzing the most common skills required by employers.
- **Employment Type**: Examining the influence of full-time, part-time, and contract work on salary.
- **Benefits**: Assessing the correlation between benefits score and job characteristics.

## File Structure

- `Global AI Job Market & Salary (1).csv`: Dataset 1 containing detailed AI job data.
- `Global AI Job Market & Salary (2).csv`: Dataset 2 with similar data from different sources.

## Requirements

The following Python libraries are required to run the analysis:
- `pandas`
- `numpy`
- `matplotlib`

You can install them using the following command:
bash
pip install pandas numpy matplotlib

## Data Analysis

The data is analyzed using basic Python techniques to:
1. Perform exploratory data analysis (EDA).
2. Generate summary statistics.
3. Visualize salary distributions, experience levels, and more.
